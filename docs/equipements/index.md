# 🛡️ Équipements & Panoplies

Les équipements améliorent tes statistiques en combat. Il existe **7 emplacements** à équiper, et des **panoplies** (sets) qui donnent des bonus supplémentaires quand plusieurs pièces du même set sont portées ensemble.

---

## Les 7 emplacements

| Emplacement | Type |
|-------------|------|
| Tête | Casque / Capuche / Couronne |
| Épaules | Épaulières / Manteau |
| Torse | Armure / Robe |
| Mains | Gantelets / Gants |
| Jambes | Jambières / Pantalon |
| Pieds | Bottes / Sandales |
| Accessoire | Anneau / Amulette / Ceinture |

---

## Les raretés

La rareté d'un équipement détermine la puissance de ses statistiques. Plus la rareté est élevée, plus l'item est puissant.

| Rareté | Puissance relative |
|--------|--------------------|
| ⬜ Commun | ×1,0 — Base |
| 🟩 Peu Commun | ×1,3 — +30% |
| 🟦 Rare | ×1,7 — +70% |
| 🟪 Épique | ×2,3 — ×2,3 |
| 🟧 Légendaire | ×3,0 — ×3 |
| 🟥 Mythique | ×4,0 — ×4 |
| 🔶 Artefact | ×5,5 — ×5,5 |
| 🟡 Divin | ×7,5 — ×7,5 |
| 🩵 Transcendant | ×10,0 — ×10 |
| 🌈 Prismatique | ×15,0 — ×15 |

---

## L'amélioration (Forgeron)

Les équipements peuvent être améliorés au Forgeron jusqu'au **niveau +20**. Chaque niveau d'amélioration augmente toutes les stats de la pièce.

---

## Les panoplies (sets)

Une panoplie est un **ensemble de 7 pièces** liées entre elles. Équiper plusieurs pièces d'une même panoplie débloque des **bonus de set** progressifs :

| Pièces équipées | Bonus débloqué |
|-----------------|---------------|
| 2 pièces | Bonus de statistiques |
| 3 pièces | Bonus passif (effet spécial) |
| 4 pièces | Bonus de statistiques |
| 5 pièces | Bonus de statistiques |
| 6 pièces | Bonus passif (effet spécial amélioré) |
| 7 pièces | Bonus passif (effet ultime de la panoplie) |

!!! info "La rareté limite les bonus"
    La pièce de **plus basse rareté** dans ton set détermine la puissance des bonus de panoplie. Une pièce commune dans un set légendaire ramènera tous les bonus au niveau commun.

---

## Les 6 sources de panoplies

Chaque classe possède **6 panoplies différentes**, une par source de jeu. Les panoplies des sources avancées sont plus puissantes mais aussi plus difficiles à obtenir.

| Source | Comment l'obtenir | Puissance |
|--------|-------------------|-----------|
| **Monde** | Drops en combattant dans les zones | Base |
| **Donjon Classique** | Drops dans les donjons classiques | +40% |
| **Craft** | Fabriquée par le Forgeron | +80% |
| **Donjon Élite** | Drops dans les donjons élite | ×2,2 |
| **Donjon Abyssal** | Drops dans les donjons abyssaux | ×3,0 |
| **Raid** | Drops dans les raids | ×4,5 |

---

## Comment optimiser son équipement ?

1. **Complète un set** avant de tout maximiser — les bonus 7 pièces sont souvent plus importants que les stats individuelles
2. **Monte la rareté** de toutes tes pièces uniformément — une pièce trop faible tire tout le set vers le bas
3. **Améliore chez le Forgeron** — même des pièces communes améliorées +20 peuvent être très efficaces
4. **Vise la source correspondant à ton niveau** — inutile de chercher des pièces de Raid si tu ne fais pas encore de Raids
