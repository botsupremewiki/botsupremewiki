# 🍞 Nourritures (Boulanger)

Les nourritures sont fabriquées par le **Boulanger**. Elles s'utilisent avant un combat pour gagner de l'énergie et/ou booster tes statistiques.

!!! info "Durée"
    Sauf indication contraire, les bonus de combat durent pour **1 combat**.

---

## Pains — Énergie directe

Les pains te donnent immédiatement de l'**énergie** quand tu les consommes.

| Pain | Énergie gagnée |
|------|---------------|
| 🍞 Pain (T1) | +10 énergie |
| 🍞 Pain (T2) | +20 énergie |
| 🍞 Pain (T3) | +35 énergie |
| 🍞 Pain (T4) | +55 énergie |
| 🍞 Pain (T5) | +75 énergie |
| 🍞 Pain (T6) | +100 énergie |

## Festins — Spéciaux haut niveau

Ces trois items combinent **énergie directe**, **bonus de régénération** et **énergie par victoire** en un seul consommable.

| Festin | Énergie | Regen bonus | Énergie / victoire |
|--------|---------|-------------|-------------------|
| 🎉 Festin de l'Aventurier (niv. 75) | +50 | +20% | +15 |
| 🏆 Banquet du Champion (niv. 90) | +80 | +35% | +25 |
| 👑 Repas Légendaire (niv. 98) | +120 | +60% | +40 |

!!! tip "Conseils"
    - Utilise une **Ration de Victoire** pour le mode Auto dans le Monde — chaque combat gagné te redonne de l'énergie.
    - Combine une nourriture avec un élixir (Alchimiste) pour maximiser tes stats avant un boss difficile.
    - Les **Festins** haut niveau sont les consommables les plus efficaces pour les joueurs avancés.
