# ⚗️ Potions & Élixirs

Les potions sont créées par le métier **Alchimiste**.

Elles se divisent en plusieurs catégories selon leur effet.


---


## 💚 Potions de Soin

| Nom | Effet |
|-----|-------|
| 🧪 **Petite Potion de Soin** | Soigne **+10%** HP max en combat |
| 🧪 **Potion de Soin** | Soigne **+20%** HP max en combat |
| 💊 **Grande Potion de Soin** | Soigne **+40%** HP max en combat |
| 💊 **Très Grande Potion de Soin** | Soigne **+60%** HP max en combat |
| ✨ **Giga Potion de Soin** | Soigne **+80%** HP max en combat |
| 🌟 **Potion de Soin Ultime** | Soigne **+100%** HP max en combat |
