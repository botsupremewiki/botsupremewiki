# 🌍 Mode Monde

Le **Monde** est le mode de jeu principal. Tu progresses zone par zone, en affrontant des ennemis de plus en plus puissants jusqu'à la zone **10 000**.


---


## ⚡ Coûts en énergie

| Type d'action | Coût ⚡ |
|---------------|--------:|
| Ennemi normal | 1 |
| Boss classique | 2 |
| Boss runique | 3 |
| Boss emblématique | 5 |
| Boss antique | 10 |


---


## 🗺️ Structure d'une zone

Chaque zone contient :

- **4 stages d'ennemis** (classes différentes selon le stage)

- **1 boss classique** pour passer à la zone suivante

- Tous les **10 zones** : un **Boss Runique** plus puissant 🔮

- Tous les **100 zones** : un **Boss Emblématique** légendaire 🌟

- Tous les **1000 zones** : un **Boss Antique**, le plus redoutable ⚠️

## 🏆 Types de boss


### ⚔️ Boss Classique — toutes les zones

Le boss de chaque zone. Il a **+20% HP** par rapport aux ennemis normaux.

Vaincre le boss permet de **passer à la zone suivante**.

- Énergie : **2 ⚡**


### 🔮 Boss Runique — toutes les 10 zones

Un boss plus puissant avec un **passif spécial** selon sa classe.

Il a **+50% HP** et utilise le **Sort 1** de sa classe.

- Énergie : **3 ⚡**


| Classe | Passif Runique |
|--------|---------------|
| ⚔️ Guerrier | **Furie Mineure** : +3% dégâts par tranche de 25% HP perdus (max +12%) |
| 🗡️ Assassin | **Réflexes** : 5% de chance d'esquiver une attaque physique |
| 🔮 Mage | **Résonance Arcanique** : Ignore 8% de ta Déf. Mag. |
| 🏹 Tireur | **Visée Précise** : +8% chance de critique |
| 🛡️ Support | **Régénération** : Récupère 0.5% HP max au début de chaque tour |
| 🧛 Vampire | **Siphon Mineur** : Récupère 5% des dégâts infligés en HP |
| ⏳ Gardien du Temps | **Distorsion Légère** : Ta vitesse réduite de 5% |
| ☠️ Ombre Venin | **Glandes Actives** : 10% de chance de poison additionnel à chaque attaque |
| 🔥 Pyromancien | **Braises** : 20% de chance d'appliquer 1 stack de brûlure à chaque attaque |
| ✝️ Paladin | **Résistance** : Dégâts reçus réduits de 5% |

### 🌟 Boss Emblématique — toutes les 100 zones

Un boss légendaire avec un **nom propre**, **+100% HP** et un passif encore plus dangereux.

Il utilise les **Sorts 1 et 2** de sa classe.

- Énergie : **5 ⚡**


Noms emblématiques (rotation) : Malachar l'Éternel, Zephyros le Tempétueux, Mordreth la Dévoreuse, Xanathos l'Omniscient, Valdris l'Implacable, Sylvara la Tisseuse, Grudnak le Colossal, Elysia la Corrompue, Tharax le Venimeux, Drakonus le Chromate...


| Classe | Passif Emblématique |
|--------|---------------------|
| ⚔️ Guerrier | **Furie Guerrière** : +5% dégâts par tranche de 25% HP perdus (max +20%) |
| 🗡️ Assassin | **Instinct** : 10% de chance d'esquiver une attaque physique |
| 🔮 Mage | **Voile Arcanique** : Ignore 12% de ta Déf. Mag. |
| 🏹 Tireur | **Œil Affûté** : +12% chance de critique |
| 🛡️ Support | **Régénération Active** : Récupère 1% HP max/tour |
| 🧛 Vampire | **Siphon de Vie** : Récupère 8% des dégâts infligés |
| ⏳ Gardien du Temps | **Ralentissement** : Ta vitesse réduite de 10% |
| ☠️ Ombre Venin | **Venin Persistant** : 15% de chance de poison additionnel à chaque attaque |
| 🔥 Pyromancien | **Embrasement** : 30% de chance d'appliquer 1 stack de brûlure à chaque attaque |
| ✝️ Paladin | **Aura Défensive** : Dégâts reçus réduits de 8% |

### ⚠️ Boss Antique — toutes les 1000 zones

Le boss ultime de chaque millénaire. **+200% HP**, passif destructeur, utilise **Sorts 1, 2 et Ultime**.

- Énergie : **10 ⚡**


Noms antiques : Vrethax le Primordial, Azkoth la Conscience Noire, Solarius le Dévoreur d'Étoiles, Mortifax l'Impérissable, Chronaxis le Maître Absolu, Nexuvor l'Indicible, Erebos la Nuit Éternelle, Pantheos le Dieu-Bête, Ultharak l'Abomination, Omegaris la Fin du Monde


| Classe | Passif Antique |
|--------|---------------|
| ⚔️ Guerrier | **Rage Ancienne** : +8% dégâts par 25% HP perdus (max +32%) + immunité aux stuns |
| 🗡️ Assassin | **Ombre Persistante** : 15% esquive + ses critiques ignorent 15% de ta Déf. Phy. |
| 🔮 Mage | **Annihilation Partielle** : Ignore 20% de ta Déf. Mag. + tous les 5 tours : dégâts purs 10% HP max |
| 🏹 Tireur | **Salve Précise** : +18% chance de critique + tes défenses physiques -10% |
| 🛡️ Support | **Bastion** : Récupère 1.5% HP max/tour + dégâts reçus -10% |
| 🧛 Vampire | **Avidité Sanguine** : Récupère 12% des dégâts infligés + te vole 1% HP max/tour |
| ⏳ Gardien du Temps | **Distorsion Temporelle** : Ta vitesse -15%, sa vitesse +15% |
| ☠️ Ombre Venin | **Nuée Venimeuse** : 20% de chance de poison additionnel + dégâts DoT ×1.2 |
| 🔥 Pyromancien | **Combustion** : 40% de chance d'appliquer 1 stack de brûlure + dégâts brûlure ×1.2 |
| ✝️ Paladin | **Jugement** : Dégâts reçus -12% + tous les 5 tours : dégâts purs 8% HP max |
