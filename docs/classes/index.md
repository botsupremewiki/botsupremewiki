# ⚔️ Les Classes

Il y a **10 classes** jouables, chacune avec un passif unique, une ressource et 3 sorts (dont un ultime).

---

## Ressources

| Ressource | Classes | Régénération |
|-----------|---------|--------------|
| **Rage** | Guerrier, Vampire, Paladin | +10/tour, +5 par coup reçu (max 100) |
| **Mana** | Mage, Pyromancien | +15/tour (max 100) |
| **Mana** | Support | +12/tour (max 100) |
| **Combo** | Assassin, Tireur, Gardien du Temps | +1/tour (max 5) |
| **Combo** | Ombre Venin | +1 par coup porté (max 5) |

## Résumé des passifs

| Classe | Nom du Passif | Description |
|--------|---------------|-------------|
| Guerrier | Rage du Blessé | +0.5% dégâts par % de HP manquants (max +30%). |
| Assassin | Ombre Furtive | 20% de chance d'esquiver une attaque ennemie. |
| Mage | Concentration Arcanique | +0.5% de dégâts magiques par % de HP actuels (max +50%). |
| Tireur | Double Tir | 25% de chance de doubler ses dégâts. |
| Support | Instinct Protecteur | 45% de chance de générer un bouclier (10% des HP max). |
| Vampire | Instinct Prédateur | 25% de chance de vol de vie (30% des dégâts infligés). |
| Gardien du Temps | Distorsion Temporelle | 55% de chance de réduire une stat ennemie aléatoire de 5% (max −20%). |
| Ombre Venin | Venin Corrosif | 30% de chance d'appliquer du poison (+35% ATK Mag par stack actif). |
| Pyromancien | Embrasement Perpétuel | 45% de chance d'appliquer 1 stack de brûlure (15% ATK Mag/stack). |
| Paladin | Bénédiction de Combat | 50% de chance d'augmenter une stat aléatoire de 5% (max +20%, permanent ce combat) — toutes les stats sauf HP. En raid, le boost s'applique à tous les alliés. |
