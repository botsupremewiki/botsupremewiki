# 📜 Quêtes

Les quêtes se déroulent en deux chaînes indépendantes accessibles depuis le **Hub Quêtes**. Dans chaque chaîne, une seule quête est active à la fois.

!!! info "Comment ça fonctionne"
    La première quête non complétée de chaque chaîne est automatiquement active. Une fois l'objectif atteint, réclame ta récompense pour passer à la suivante.

---
